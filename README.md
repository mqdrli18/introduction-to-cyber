📌 Overview
This repository provides essential knowledge about how systems, networks, and data are protected.  
It is designed as a starting point for anyone interested in cybersecurity.
🧠 Topics Covered
- 🔑 Encryption (Symmetric & Asymmetric)  
- 🔐 Hashing & Digital Signatures  
- 🌐 Network Security Basics  
- 🧱 Firewalls & Access Control  
- 🔄 TCP vs UDP  
- 🖥️ Client-Server vs P2P  
- ☁️ Cloud Basics
